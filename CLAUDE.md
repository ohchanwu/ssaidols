# LocalHub — SSAFY 3일차 팀프로젝트

## AI 도구 정책 (중요)

원본 개발 의뢰서(RFP) III-7 항목은 Claude Code 사용을 금지하고 VSCode Copilot + OpenAI API만 허용한다.
**단, 담당 교수가 이 프로젝트에 한해 Claude Code 사용을 허용했다 (사용자 확인).**
따라서 이 저장소에서는 Claude Code를 사용해도 되며, 매 세션마다 이 예외를 다시 확인할 필요는 없다.

주의: 이 예외는 *도구 사용*에 대한 것이다. 산출물(실습기획서·기능명세서)에 기재하는 **제약사항 항목에는
RFP 원문대로 "VSCode Copilot + OpenAI API만 허용"을 그대로 적어야 한다.** 문서의 내용을 바꾸는 예외가 아니다.

## 기술 제약 (RFP 고정 — 임의 변경 금지)

- **백엔드 서버 없음.** Vue.js 3 (Vite) 정적 SPA 단일 구조.
- 게시글은 **브라우저 localStorage**에 저장. 수정용 비밀번호는 **평문 저장·비교** (교육 목적 의도된 설계).
- 회원가입·로그인 없음. 인증/권한 체계 없음.
- 챗봇은 프론트엔드에서 **OpenAI API를 직접 호출**. 키는 `.env`의 `VITE_` 접두사로 관리.
  - `VITE_` 값은 빌드 결과물에 노출된다. 사용량 제한이 걸린 키만 사용할 것.
- 배포는 **Netlify만** (Render 등 백엔드 배포 없음).
- `.env`는 절대 커밋하지 않는다. `.gitignore` 등록 필수.

> 참고: 원본 배포판에 함께 들어 있던 `작성예제.xlsx`는 **FastAPI + SQLite 백엔드가 있는 다른 버전의 과제**를
> 기준으로 작성된 샘플이었다. 구조·톤만 참고하고 기술 내용은 절대 따라 쓰지 말 것. (현재는 저장소에서 삭제됨)

## 데이터 실측 정보 (`서울_data/` — 반드시 이 값을 기준으로 판단할 것)

한국관광공사 TourAPI 4.0 / **공공누리 제3유형** (출처 표시 필수, 변경 금지, 상업적 이용 허용).
앱에 "한국관광공사" 및 원본 API URL 표기 의무.

실제 파일 7개, **총 6,518건 / 5.61MB**. 필드 정의는 `서울_data/SCHEMA.md`.

**SOURCE.md와 실제 파일이 불일치하는 부분이 있으므로 아래 실측값을 신뢰할 것:**

- ❌ **`서울_음식점.json` 파일이 존재하지 않는다.** SOURCE.md는 1,632건이 있다고 기재하고 있으나 디스크에 없다.
  RFP가 챗봇 질의 유형으로 명시한 "모범음식점 위치"는 제공 데이터로 응답 불가. 맛집 기능은 전제하지 말 것.
- ❌ **축제 데이터에 일정(날짜) 필드가 없다.** `eventstartdate`/`eventenddate` 부재.
  `createdtime`/`modifiedtime`만 존재하므로 **"축제 캘린더" 선택기능은 제공 데이터만으로 구현 불가.**
- ⚠️ `tel` 필드는 **3.1%만 채워져 있다.** 지도 팝업·상세정보에 전화번호를 넣지 말 것.
- ⚠️ `cat1`/`cat2`/`cat3`은 **22%만 채워져 있고**, 코드표(`lclsSystemCode.json`)도 저장소에 없다.
  **카테고리 필터는 `lclsSystm1`/`lclsSystm2`(100% 충족) 또는 `contenttypeid`를 사용할 것.**
- ✅ `title` 100%, `mapx`/`mapy` 100%, `addr1` 99.2%, `firstimage` 94.3% — 지도·목록·이미지는 안전.
- `mapx`(경도)/`mapy`(위도)는 **string**이므로 숫자 변환 필요.

파일별 건수: 쇼핑 4,368 / 관광지 783 / 문화시설 566 / 숙박 423 / 축제공연행사 201 / 레포츠 126 / 여행코스 51.
`서울_쇼핑.json` 하나가 3.9MB(전체의 69%)다. 전량 eager-load 금지.

- 원본 JSON은 **변경 금지**(공공누리 3유형). 가공이 필요하면 빌드 단계에서 파생 파일을 만들 것.
- 챗봇에 6,518건을 통째로 넣지 말 것. 브라우저에서 먼저 필터링해 상위 소수 건만 컨텍스트로 전달한다.
